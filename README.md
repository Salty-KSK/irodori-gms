# GMS - カメラマン総合管理システム

📷 カメラマンのための総合業務管理Webアプリケーション

## 機能一覧

| ページ | 機能 |
|--------|------|
| ダッシュボード | 売上/経費サマリー、今週の予定、アラート |
| 顧客リスト | 顧客管理（CRUD、検索、フィルター） |
| 案件リスト | 案件管理（テーブル/カンバン、同意チェック） |
| 収支管理 | 売上・経費管理、グラフ表示 |
| 機材リスト | 機材・メンテナンス履歴管理 |
| 仕入れ先 | 仕入先管理（評価付き） |
| ロケ地DB | ロケ地情報・撮影ヒント管理 |
| SNSネタ帳 | コンテンツアイデア管理（カンバン） |
| Instagram保存 | 参考投稿ギャラリー |
| サブスク管理 | サブスクリプション・コスト管理 |
| 予約管理 | 予約フロー管理 |

## 技術スタック

- HTML5 / CSS3 / JavaScript (ES Modules)
- Material Design 3 準拠 UI
- LocalStorage (Phase 1) → Firebase (Phase 3)

## 開発

```bash
# ローカルサーバー起動
python -m http.server 8080
```

http://localhost:8080 でアクセス
